# SentinelZero -- Continuous Post-Audit Risk Monitoring for DeFi

## Overview
SentinelZero provides continuous post-audit monitoring for DeFi protocols, analyzing historical exploits, generating risk scores, and providing actionable insights for developers and investors.

## Repository Structure
- `docs/` -- Detailed project documentation
- `src/` -- Python scripts and APIs
- `tests/` -- Unit and integration tests
- `assets/` -- Images, mockups, and diagrams
